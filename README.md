# MinecraftTechLauncher简介
![icon](https://user-images.githubusercontent.com/111875719/245127705-519e6b9d-9583-42e1-a9f1-361ce90b6dec.png)

**B:因为启动器基于图标文件或者配置文件?(滑稽)** <br>
